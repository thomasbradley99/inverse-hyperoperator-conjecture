# Conjecture

This folder contains the concise argument and proof roadmap.

## Files

- `conjecture.tex`: main write-up (with computational evidence summary).
- `conjecture.pdf`: compiled version of the write-up.
- `PROOF_STEPS.md`: plain-language theorem roadmap.

## Build

From this folder:

```bash
pdflatex -interaction=nonstopmode conjecture.tex
```

## Scope

This is an evidence-first outline: strong computational support, explicit limits,
and clear next steps toward formal proof.
