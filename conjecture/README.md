# Conjecture

The conjecture statement, motivation, and concrete prediction — no experimental results.

## Files

- `conjecture.tex`: the Inverse Hyperoperator Information Conjecture and its prediction for inverse `z^z`.
- `PROOF_STEPS.md`: plain-language theorem roadmap.

## Build

```bash
pdflatex -interaction=nonstopmode conjecture.tex
```

## Scope

This folder states the conjecture only. Computational evidence is in `../exploration/`.
